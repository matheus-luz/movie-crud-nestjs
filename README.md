# Orientações

Documentação [NestJs](https://github.com/nestjs/nest):

## Instalação

```bash
npm install
```

## Rodando o App

```bash
# docker postgres
npm run compose:up

# development
npm run start

# watch mode
npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
